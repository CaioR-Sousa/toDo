# ✅ ToDo List

Uma aplicação web desenvolvida para organizar tarefas do dia a dia de forma simples, prática e intuitiva.

## 📖 Sobre o projeto

O **ToDo List** foi desenvolvido com o objetivo de praticar os fundamentos do Desenvolvimento Front-End, permitindo ao usuário criar, gerenciar e acompanhar suas tarefas.

Além de oferecer uma interface limpa e organizada, o projeto aplica conceitos importantes de manipulação do DOM, eventos e armazenamento local utilizando JavaScript.

## ✨ Funcionalidades

* ➕ Adicionar novas tarefas
* ✅ Marcar tarefas como concluídas
* ✏️ Editar tarefas existentes
* 🗑️ Remover tarefas
* 📋 Exibir todas as tarefas cadastradas
* 💾 Salvar automaticamente as tarefas utilizando **LocalStorage**
* 🔄 Manter os dados após atualizar ou fechar o navegador

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage
